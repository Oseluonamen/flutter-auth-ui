Project title: Flutter Authentication UI

Description:
A clean and responsive Flutter authentication UI featuring login, signup, and forgot password screens with form validation and smooth navigation.

Features:
- Login, Signup, and Forgot Password screens
- Form validation
- Loading states
- Clean navigation
- Responsive UI
